# Proyecto Fase 4 - Sistema de Software

## Resumen Ejecutivo

### Descripción
Este proyecto consiste en el desarrollo de un sistema de software que permite gestionar funcionalidades básicas mediante una arquitectura cliente-servidor.

### Problema identificado
Las organizaciones requieren herramientas digitales que permitan administrar procesos de manera eficiente, sin contar con sistemas estructurados.

### Solución
Se desarrolló una solución basada en tecnologías modernas, utilizando control de versiones, integración continua y gestión de tareas.

### Arquitectura
El sistema sigue una arquitectura cliente-servidor, donde el usuario interactúa mediante un navegador, el servidor procesa la lógica y la base de datos almacena la información.
